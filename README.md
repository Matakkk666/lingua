# AI English School

> Онлайн-платформа для изучения английского языка с геймификацией, личным прогрессом и выдачей сертификатов.

🌐 **Живой сайт:** [durifiri.eu.cc](https://durifiri.eu.cc)

---

## О проекте

AI English School — это учебная платформа с бесплатными и платными курсами английского. Пользователь регистрируется, проходит уроки с интерактивными тестами, отслеживает свой прогресс и получает сертификат по завершении.

Проект сделан как законченный продукт: с авторизацией, оплатой, юридическими документами и адаптацией под мобильные устройства.

## Возможности

- 🔐 **Авторизация и аккаунты** — регистрация и вход через Supabase.
- 📚 **Курсы** — бесплатные и платные, с пошаговым прохождением уроков.
- 📝 **Интерактивные тесты** с мгновенной проверкой ответов.
- 🔥 **Геймификация** — стрики (streak) и ежедневные цели для удержания мотивации.
- 📊 **Дашборд прогресса** — визуализация успехов пользователя.
- 🏆 **Сертификаты** по завершению курса.
- 💳 **Оплата** платных курсов через Robokassa.
- 📱 **Адаптивный дизайн** — отдельная оптимизация под телефоны.
- ⚖️ **Юридические страницы** — публичная оферта и политика конфиденциальности.

## Стек технологий

| Категория | Технологии |

## Запуск локально

Проект статический — сборка не нужна. Достаточно локального сервера:

```bash
# Клонировать репозиторий
git clone https://github.com/Matakkk666/lingua.git
cd lingua

# Запустить локальный сервер (любой на выбор)
python -m http.server 8000
# или
npx serve
```

Затем открыть `http://localhost:8000` в браузере.

Для полной работы авторизации и оплаты настройте ключи по инструкциям в [SETUP-AUTH.md](SETUP-AUTH.md) и [SETUP-PAYMENT.md](SETUP-PAYMENT.md).
